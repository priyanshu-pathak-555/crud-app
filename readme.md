This project is about user authentication system using flask  and sqlite. it allows  user to to register , log in and securely  acces a protected dashboard 
using session- based authentication , the application ensures  that only authentivcated  users can  acces specific routes  , making it a practical implemnetation  of  real  world  login system

The system follows secure development practices by using password hashing instead of storing plain text passwords. This enhances data security and aligns with industry standards. The backend is built using Flask, while SQLite is used for efficient and lightweight database management.

In addition to authentication, the project demonstrates full CRUD (Create, Read, Update, Delete) operations through structured data handling, making it a complete full-stack application. The frontend is designed using HTML and CSS to provide a clean and user-friendly interface.
